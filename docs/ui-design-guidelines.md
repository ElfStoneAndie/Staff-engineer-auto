# UI Design Guidelines

This document outlines the UI design principles and compliance requirements for the Staff Engineer Auto project, specifically for integration with Apple CarPlay.

## Guiding Principles
1. Simplicity: The interface should be user-friendly and distraction-free.
2. Legibility: Ensure text and interfaces are easy to read under various lighting conditions.
3. Accessibility: Support voice input and touch gestures.

## Apple CarPlay Compliance
- Adhere to Apple's Human Interface Guidelines for CarPlay.
- Minimize user interaction while driving.
